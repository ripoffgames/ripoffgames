Instructions for Closed Beta Testing

Requirements
  • iOS 9.0 or later
  • iPhone 5 or later
  • Ability to download apps
  • Apple ID
  • Ability to check email
Instructions
  1. Download the TestFlight app
  2. Fill out the closed beta test submission form here
  3. Check email to receive validation code
  4. Open TestFlight
  5. Sign in with Apple ID
  6. Click the button labeled "Redeem"
  7. Enter validation code from email
  8. Start Testing!
